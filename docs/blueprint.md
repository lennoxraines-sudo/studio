# **App Name**: Arcade Escape

## Core Features:

- Game Browse and Listing: Display a browsable catalog of available unblocked games, fetched from a local JSON file.
- Game Search and Filter: Allow users to quickly find games using a search bar or by applying basic filters.
- In-Game Iframe Player: Load and embed the selected game dynamically within an iframe on the page for interactive play.
- Game Details View: Show essential information for each game, such as title, and a brief description.
- Fullscreen Game Toggle: Provide a button to expand the game iframe to occupy the full screen for an immersive experience.
- Responsive Design: Ensure the application's layout and content are fully responsive and functional across various device screen sizes.

## Style Guidelines:

- Primary color: A vibrant violet (#6133CC) chosen for its playful yet sleek digital feel, hinting at nostalgic arcade lights and modern tech.
- Background color: A subtle, desaturated lilac (#EFECF5), providing a clean, light canvas that complements the primary color without being distracting.
- Accent color: A brighter blue-violet (#6E88FF) to draw attention to interactive elements and highlights, offering a refreshing contrast to the primary.
- Headline and body font: 'Space Grotesk' (sans-serif), chosen for its modern, tech-inspired, and slightly condensed aesthetic, perfect for game titles and concise descriptions.
- Utilize simple, geometric icons for navigation, search, and game controls that align with the digital and 'unblocked' theme. Examples include a play button, a search magnifying glass, and a fullscreen icon.
- Implement a clean, grid-based layout for game listings to ensure easy browsing. The game play area should dominate the screen when a game is active, with minimal surrounding UI.
- Incorporate subtle hover effects on game cards and smooth transitions when opening a game, enhancing interactivity without being overly complex for an MVP.
